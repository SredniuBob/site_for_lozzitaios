Быстрый старт с LozzitaiOS

LozzitaiOS 24.04.3 — это современный дистрибутив Linux, основанный на Ubuntu 24.04.3 LTS Noble Numbat. Система предлагает стабильность LTS-релиза с актуальными обновлениями безопасности и программного обеспечения.

Ключевые особенности версии 24.04.3:

  • Ядро: Linux 6.8+ с улучшенной поддержкой оборудования
  • Графический стек: Wayland по умолчанию для лучшей безопасности
  • Python 3.12, GCC 13, GNOME 46 в стандартной поставке
  • Полная поддержка ARM64 архитектуры (Raspberry Pi 4/5)
  • Долгосрочная поддержка до апреля 2029 года

С чего начать?

  1. Проверьте системные требования в соответствующем разделе
  2. Скачайте образ подходящей архитектуры (x86_64 или ARM64)
  3. Создайте загрузочный USB с помощью Rufus, BalenaEtcher или dd
  4. Загрузитесь с USB и выберите "Try LozzitaiOS" для тестирования или "Install LozzitaiOS" для установки
  5. Следуйте инструкциям установщика - процесс интуитивно понятен

Минимальные требования для x86_64

Требования для ARM64 (Raspberry Pi)

Поддерживаемые устройства:

  • Raspberry Pi 4 (все версии, 2+ ГБ ОЗУ)
  • Raspberry Pi 5 (рекомендуется 4+ ГБ ОЗУ)
  • Orange Pi 5/5B
  • Rock Pi 4/5
  • Другие устройства с Cortex-A72/A76/A78
Устройство ОЗУ Хранилище Особенности
Raspberry Pi 4 2+ ГБ 32+ ГБ microSD USB boot, PCIe через USB3
Raspberry Pi 5 4+ ГБ SSD через PCIe или microSD Рекомендуется NVMe через HAT
Orange Pi 5 8+ ГБ NVMe или eMMC Лучшая производительность

Важно: Для Raspberry Pi рекомендуется использовать карты памяти Class 10/U3 или SSD через USB3 для лучшей производительности.

ARM64 Версия для одноплатных компьютеров

LozzitaiOS ARM64 Edition специально оптимизирован для работы на одноплатных компьютерах, таких как Raspberry Pi, Orange Pi и других ARM-устройствах.

Особенности ARM64 версии

  • Оптимизированное ядро с поддержкой всех функций Raspberry Pi
  • Аппаратное ускорение для графики и видео (VC4/V3D)
  • Поддержка GPIO и различных датчиков через библиотеки Python
  • Энергоэффективная работа с управлением питанием
  • Предустановленные инструменты для IoT и встраиваемых систем

Установка на Raspberry Pi

$ lsblk
# Определите имя вашего USB накопителя (обычно sdb или sdc)

$ sudo dd if=lozzitaios-arm64-24.04.3.img of=/dev/sdX bs=4M status=progress
# Замените sdX на ваше устройство (например, sdb)

$ sudo sync
# Дождитесь завершения записи

Настройка после установки

# Расширить файловую систему на всю карту
sudo raspi-config --expand-rootfs

# Включить SSH для удалённого доступа
sudo systemctl enable ssh
sudo systemctl start ssh

# Настроить локализацию
sudo raspi-config
# Выберите: Localisation Options → Locale → ru_RU.UTF-8

# Обновить систему
sudo apt update && sudo apt upgrade -y

Оптимизация производительности для Raspberry Pi

# Настроить файл подкачки (swap) для Pi с малым объёмом ОЗУ
sudo dphys-swapfile swapoff
sudo nano /etc/dphys-swapfile
# Измените CONF_SWAPSIZE=2048 (для 4ГБ ОЗУ)
sudo dphys-swapfile setup
sudo dphys-swapfile swapon

# Отключить ненужные сервисы для экономии ресурсов
sudo systemctl disable bluetooth.service
sudo systemctl disable hciuart.service

# Оптимизировать настройки файловой системы
sudo nano /etc/fstab
# Добавьте: noatime,nodiratime к опциям монтирования корневого раздела

# Установить ZRAM для сжатия оперативной памяти
sudo apt install zram-tools
sudo systemctl enable zramswap

Совет: Для Raspberry Pi 4/5 рекомендуется использовать активное охлаждение и качественный блок питания (5V/3A) для стабильной работы.

Управление пакетами в LozzitaiOS

LozzitaiOS использует систему управления пакетами APT от Ubuntu с поддержкой всех форматов пакетов.

Основные команды APT

$ sudo apt update
# Обновить списки пакетов из репозиториев

$ sudo apt upgrade
# Обновить все установленные пакеты

$ sudo apt full-upgrade
# Полное обновление с возможным удалением пакетов

$ sudo apt install package_name
# Установить пакет

$ sudo apt remove package_name
# Удалить пакет, оставив конфигурационные файлы

$ sudo apt purge package_name
# Полностью удалить пакет с конфигами

$ sudo apt autoremove
# Удалить неиспользуемые зависимости

$ apt search keyword
# Найти пакет по ключевому слову

$ apt show package_name
# Показать информацию о пакете

Управление репозиториями

# Добавить PPA репозиторий
sudo add-apt-repository ppa:repository/name

# Удалить PPA репозиторий
sudo add-apt-repository --remove ppa:repository/name

# Добавить репозиторий через sources.list
echo "deb http://archive.ubuntu.com/ubuntu noble main" | sudo tee /etc/apt/sources.list.d/custom.list

# Добавить ключ репозитория
wget -qO - https://keyserver.ubuntu.com/pks/lookup?op=get&search=0xKEY_ID | sudo apt-key add -

# Обновить GPG ключи
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys KEY_ID

Работа с deb-пакетами

# Установить deb-пакет локально
sudo dpkg -i package.deb

# Установить с автоматическим разрешением зависимостей
sudo apt install ./package.deb

# Просмотреть содержимое deb-пакета
dpkg -c package.deb

# Проверить установленные пакеты
dpkg -l | grep package_name

# Найти, какому пакету принадлежит файл
dpkg -S /path/to/file

Snap и Flatpak пакеты

LozzitaiOS поддерживает все современные форматы пакетов:

# Установка Snap (предустановлен)
sudo snap install package-name

# Установка Flatpak
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub org.app.ID

# Просмотр установленных Snap
snap list

# Просмотр установленных Flatpak
flatpak list

Настройка сети и подключений

Базовая настройка сети

# Просмотр сетевых интерфейсов
ip addr show
nmcli device status

# Настройка статического IP (через Netplan)
sudo nano /etc/netplan/01-netcfg.yaml
# Добавьте конфигурацию:
network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      addresses:
        - 192.168.1.100/24
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]

# Применить конфигурацию
sudo netplan apply

Wi-Fi подключение через терминал

# Просмотр доступных Wi-Fi сетей
nmcli device wifi list

# Подключение к Wi-Fi
nmcli device wifi connect "SSID" password "password"

# Подключение с сохранением
nmcli connection add type wifi con-name "HomeWiFi" ifname wlan0 ssid "SSID"
nmcli connection modify "HomeWiFi" wifi-sec.key-mgmt wpa-psk wifi-sec.psk "password"
nmcli connection up "HomeWiFi"

# Автоматическое подключение при загрузке
nmcli connection modify "HomeWiFi" connection.autoconnect yes

Настройка DNS

# Изменение DNS серверов
sudo nano /etc/systemd/resolved.conf
# Раскомментируйте и измените:
DNS=8.8.8.8 8.8.4.4 1.1.1.1
FallbackDNS=9.9.9.9 149.112.112.112

# Применить изменения
sudo systemctl restart systemd-resolved

# Временное изменение DNS
sudo resolvectl dns eth0 8.8.8.8 1.1.1.1

Основы безопасности системы

Базовая настройка безопасности

# Обновить систему
sudo apt update && sudo apt upgrade -y

# Установить базовые инструменты безопасности
sudo apt install fail2ban rkhunter chkrootkit unattended-upgrades

# Настроить автоматические обновления безопасности
sudo dpkg-reconfigure -plow unattended-upgrades
sudo nano /etc/apt/apt.conf.d/50unattended-upgrades
# Раскомментируйте строки с обновлениями безопасности

Настройка фаервола UFW

# Установить и включить UFW
sudo apt install ufw
sudo ufw default deny incoming
sudo ufw default allow outgoing

# Разрешить основные сервисы
sudo ufw allow ssh
sudo ufw allow http
sudo ufw allow https

# Разрешить конкретные порты
sudo ufw allow 22/tcp
sudo ufw allow 80,443/tcp
sudo ufw allow from 192.168.1.0/24 to any port 22

# Включить фаервол
sudo ufw enable
sudo ufw status verbose

Настройка AppArmor

# Проверить статус AppArmor
sudo aa-status

# Включить/выключить профиль
sudo aa-enforce /etc/apparmor.d/usr.bin.firefox
sudo aa-complain /etc/apparmor.d/usr.bin.firefox

# Обновить профили AppArmor
sudo apparmor_parser -r /etc/apparmor.d/*

# Просмотр логов AppArmor
sudo journalctl -u apparmor --since "1 hour ago"

Настройка веб-сервера

# Установить Nginx
sudo apt install nginx
sudo systemctl enable nginx
sudo systemctl start nginx

# Установить Apache
sudo apt install apache2
sudo systemctl enable apache2
sudo systemctl start apache2

# Установить PHP
sudo apt install php-fpm php-mysql php-curl php-gd php-mbstring php-xml php-zip

# Настройка виртуальных хостов
sudo nano /etc/nginx/sites-available/example.com
# Добавьте конфигурацию:
server {
    listen 80;
    server_name example.com www.example.com;
    root /var/www/example.com;

sudo ln -s /etc/nginx/sites-available/example.com /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

Часто задаваемые вопросы

Как обновить LozzitaiOS до новой версии?

Для обновления с версии на базе Ubuntu 22.04:

# Сделать резервную копию важных данных
# Обновить текущую систему
sudo apt update && sudo apt upgrade -y

# Установить утилиту обновления
sudo apt install update-manager-core

# Запустить обновление до 24.04
sudo do-release-upgrade

# Следовать инструкциям на экране

Проблемы с драйверами NVIDIA на ARM64

# Для ARM64 устройств с GPU NVIDIA
sudo apt install nvidia-jetpack
sudo apt install nvidia-l4t-core

# Настройка драйверов для Jetson
sudo /usr/sbin/nvpmodel -m 0  # Максимальная производительность
sudo jetson_clocks           # Разблокировка частот

Как восстановить загрузчик GRUB?

# С Live USB запустить терминал
sudo mount /dev/sdXY /mnt
sudo mount /dev/sdXZ /mnt/boot/efi

# Перемонтировать системные каталоги
sudo mount --bind /dev /mnt/dev
sudo mount --bind /dev/pts /mnt/dev/pts
sudo mount --bind /proc /mnt/proc
sudo mount --bind /sys /mnt/sys

# Войти в chroot
sudo chroot /mnt

# Переустановить GRUB
grub-install /dev/sdX
update-grub
exit
sudo reboot

Оптимизация для игр на ARM64

# Для Raspberry Pi 4/5
# Установить Mesa с поддержкой Vulkan
sudo apt install mesa-vulkan-drivers

# Настроить производительность GPU
sudo nano /boot/firmware/config.txt
# Добавить:
gpu_mem=256    # Для Pi 4 с 4ГБ ОЗУ
gpu_mem=512    # Для Pi 5 с 8ГБ ОЗУ
over_voltage=2
arm_freq=1800  # Для Pi 4
arm_freq=2400  # Для Pi 5
