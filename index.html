Что нового в 24.04.3?

Ключевые улучшения и нововведения на базе Ubuntu Noble Numbat

Производительность

Оптимизированное ядро Linux 6.8 с улучшенной поддержкой современных процессоров и графики. Более быстрая загрузка и отклик системы.

Безопасность

Обновлённые механизмы безопасности: улучшенный AppArmor, поддержка TPM 2.0, Secure Boot по умолчанию, изолированные среды исполнения.

Графический стек

Wayland по умолчанию, поддержка HDR, улучшенная работа с несколькими мониторами, новая версия GNOME 46 с нашими модификациями.

Поддержка ARM64

Полноценная версия для Raspberry Pi 4/5, Orange Pi, и других ARM64 устройств. Оптимизированная работа на энергоэффективных платформах.

Для разработчиков

Python 3.12, GCC 13, Ruby 3.2, Go 1.22, Node.js 20, последние версии Docker и Podman. Полный стек современных инструментов разработки.

Игры и мультимедиа

Улучшенная поддержка Steam, Wine 9.0, последние версии Mesa для игр. Поддержка AV1 кодирования и декодирования аппаратно.
